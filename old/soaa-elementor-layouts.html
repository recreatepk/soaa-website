Our Story

Who We Are

Sindh Outdoor Advertisers Association (SOAA) is the only representative trade body for media owners operating in Out-of-Home Advertising in the province of Sindh. Established to give a unified voice to the industry, SOAA has grown to represent over 120 member companies, collectively managing thousands of billboard and OOH sites across Karachi, Hyderabad, and surrounding districts.

Our Mission

Unite media owners, advertisers, and government to foster a collaborative, future-ready OOH ecosystem driven by technology, innovation, and transparent governance.

Our Vision

Elevate Sindh OOH landscape and unlock the industry's full potential through data-driven environments, strong advocacy, and an empowered membership community.

WordPress Setup Guide

Step-by-step installation and configuration plan for the SOAA website.

Phase 1: Hosting & WordPress Installation

  1. Choose hosting: Cloudways (DigitalOcean 4GB) — best performance for WordPress in Pakistan region
  2. Install WordPress 6.5+ via one-click installer
  3. Set up SSL certificate (free via Let's Encrypt)
  4. Configure www.soaa.pk DNS pointing to server
  5. Set permalink structure: Post name (/%postname%/)
  6. Set timezone to Karachi (UTC+5)

Phase 2: Theme Setup (Kadence)

  1. Install Kadence Theme (free) + Kadence Blocks Pro or Elementor Pro
  2. Import a blank/minimal Kadence starter template as base
  3. Configure Global Colors in Kadence: Navy #0A1628, Blue #1B4FD8, Gold #C8922A
  4. Set Global Fonts: Playfair Display (headings) + DM Sans (body) via Google Fonts
  5. Build sticky header using Kadence Header Builder: Logo | Nav Menu | CTA Button
  6. Build footer with 4-column layout using Kadence Footer Builder
  7. Enable transparent header on Homepage template only

Phase 3: Plugins Installation Order

  1. CPT UI → Register all 4 custom post types first
  2. Advanced Custom Fields Pro → Create all field groups (see CPT tab)
  3. Elementor Pro → Activate, connect license
  4. RankMath SEO → Setup wizard, enable Schema for Events + Articles
  5. WPForms Pro → Create Contact form + Membership Application form
  6. WP Rocket / LiteSpeed Cache → Configure after all content is live
  7. Smush Pro or ShortPixel → Image optimization

Phase 4: Page Creation & Elementor Layouts

  1. Create all pages in WordPress (see sitemap): Home, About, What We Do (×3), Membership (×2), Events, Research, Publications, Members, Contact
  2. Assign custom page templates in Elementor: Full-Width (no sidebar)
  3. Build Homepage first using sections shown in Wireframe tab
  4. Use Elementor Theme Builder to create: Single Event template, Single Research template, Events Archive, Research Archive
  5. In Theme Builder → Loop Item → create Event card template (used in homepage + archive)
  6. Set up Navigation Menus: Primary Menu (with dropdowns), Footer Menu, Mobile Menu
  7. Add breadcrumbs via RankMath breadcrumb widget in Elementor

Phase 5: SEO & Performance

  1. Configure RankMath: Set H1 = Post/Page Title only (never use H1 in content)
  2. Enable LocalBusiness schema on Contact page (address, phone, hours)
  3. Enable Event schema on Events CPT (auto-populates from ACF fields)
  4. Set up XML sitemap → submit to Google Search Console
  5. Configure Open Graph for social sharing (Facebook, WhatsApp previews)
  6. Optimize images: max 200KB for hero, 80KB for thumbnails. Use WebP format.
  7. Enable lazy loading for all images
  8. Set up Google Analytics 4 via Site Kit by Google plugin

Phase 6: Admin Training & Scalability

For non-technical admins to easily manage content:

  • Create custom Admin role "Content Editor" with restricted capabilities
  • All CPTs are manageable from WordPress Admin sidebar without touching Elementor
  • Use Admin Columns Pro for searchable, sortable event/member lists in admin
  • Add a "Quick Reference" admin page using Admin Notices Manager
  • Future Scalability: Add Member Login Portal using Ultimate Membership Pro or MemberPress
  • Future: DOOH industry map using Google Maps API + ACF Coordinates field
  • Future: Job board CPT for industry classifieds
  • Future: E-commerce for event tickets via WooCommerce add-on
